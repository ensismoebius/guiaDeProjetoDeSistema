\resumo{
Este manual apresenta um guia completo para execução de um projeto de software
em quatro meses, com turmas iniciantes de vinte alunos.  
Inclui práticas técnicas, metodologias de desenvolvimento, estratégias de
organização, engajamento baseado em evidências científicas e orientações
operacionais detalhadas.  
O objetivo é fornecer uma estrutura prática, acessível e altamente replicável
para cursos técnicos e de graduação.
}
